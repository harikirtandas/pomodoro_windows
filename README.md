# ⏱️ Temporizador Pomodoro - Versión Windows (Python + Tkinter)

Temporizador de escritorio minimalista desarrollado en Python con Tkinter, basado en la técnica Pomodoro: 25 minutos de trabajo seguidos por 5 minutos de descanso.

Esta versión está optimizada para funcionar correctamente en **Windows**, utilizando la librería `playsound` para reproducir alertas sonoras.

---

## ✅ Funcionalidades

- Cuenta regresiva visual en pantalla.
- Alertas de sonido al finalizar cada ciclo (compatible con Windows).
- Interfaz gráfica simple, intuitiva y sin distracciones.
- Posibilidad de repetir los ciclos de trabajo y descanso.
- Hilos (`threading`) para evitar que la interfaz se congele.

---

## 🚀 Cómo ejecutar

1. Cloná este repositorio:
   bash
   git clone https://github.com/harikirtandas/pomodoro_windows.git
   cd pomodoro_windows

2. pip install playsound
3. python temporizador.py

📦 Requisitos
	•	Python 3.x
	•	Sistema operativo Windows
	•	Librerías:
	•	tkinter (incluida en Python)
	•	playsound (instalable con pip)

⸻

📚 Estructura del código
	•	temporizador.py: script principal con lógica del temporizador y la interfaz.
	•	sonido.mp3: archivo de sonido que se reproduce al terminar cada ciclo.
